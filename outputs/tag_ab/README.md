# Paralinguistic Tag A/B Test

Generated by `test_tags_ab.py`. Each tag is run through both models using bundled default voices (no `audio_prompt_path`).

## Folders
- `baseline/` — same sentence, no tags, to calibrate each model's default voice.
- `turbo_native/` — tags in Turbo's vocab (IDs 50257–50275). Cross-tested on Standard to hear char-split OOV behavior.
- `standard_native/` — tags in Standard's vocab (IDs 604+). Cross-tested on Turbo.
- `shared_renamed/` — same concept, different surface form per model (e.g. Turbo `[whispering]` vs Standard `[whisper]`).

## Filename convention
`<concept>__<model>[_tagform].wav` — pair up by concept prefix.

See `tokenization_report.txt` for per-prompt token IDs.
